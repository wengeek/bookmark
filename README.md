# bookmark

演示地址：[https://bm.wenjs.me](https://bm.wenjs.me)

### 概述
前端采用React＋Redux的方式，结合webpack进行开发，后端采用Express + MongoDB

### 安装mongodb
Mac下，采用brew install mongodb

### 启动mongodb(需先创建/data/db/mongodb)
mongod --dbpath /data/db/mongodb &

### npm start
启动调试服务

### npm build
执行编译

### npm run prod
执行编译并启动服务
